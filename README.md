# JAVEN & CO. - Premium D2C Ecommerce Website

A premium, responsive ecommerce website for the fashion brand JAVEN & CO., built with React, Vite, TypeScript, Tailwind CSS, and Framer Motion.

## Tech Stack

- **React 18**
- **Vite** (Build tool)
- **TypeScript**
- **React Router** (Client-side routing)
- **Tailwind CSS**
- **Framer Motion**

## Features

- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Premium minimal UI/UX
- ✅ Smooth animations with Framer Motion
- ✅ Loading screen with cinematic effects
- ✅ Product listing with filters
- ✅ Product detail pages
- ✅ Shopping cart
- ✅ Checkout flow
- ✅ About and Contact pages

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser (Vite default port).

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
